什麼是 LeetCode

LeetCode 是一個程式語言面試練習平台，幫助人們準備科技職業，被公司如 Google 用來評估程式技能。學生與專業人士也用它來合作和增進技術。 Leetcode 有簡單、中、難 3 種題型，想看更多題可以去 Leetcode 競賽

LeetCode 競賽

面試中常見的兩個 Leetcode 學習計畫是 Top Interview 150 和 Leetcode 75。 參加 Leetcode 競賽可證明編碼能力並贏得積分，積分可兌換30天 Leetcode Premium 解鎖更多問題。Leetcode 舉辦兩種主要比賽：每週比賽一週一次，雙週比賽每兩週一次。
